This page demonstrates the hover preview functionality for Obsidian-like double bracket links.

Regular Internal Links

Here's a link to Index and another to Core Rules, How to Play.

Bracket Syntax

The system will also detect and convert links written with the double bracket syntax like [[index]] and [[core-rules,-how-to-play]]. These will be automatically converted to hoverable links.

Mixed Content

You can mix regular text with [[index]] links in the same paragraph. This paragraph contains a regular link and a [[core-rules,-how-to-play]] bracket link.

Non-existent Pages

Links to non-existent pages like [[this-page-does-not-exist]] will show an error in the preview.
